# Text-to-SQL Chatbot

The **Text-to-SQL Chatbot** is an AI-powered application that allows users to query databases using **natural language instead of writing SQL queries**. The system leverages **Large Language Models (LLMs)** to translate user questions into SQL statements and retrieves relevant results from a **MySQL database**.

This project helps bridge the gap between **non-technical users and data access**, enabling team members such as analysts, managers, or business users to retrieve insights without needing SQL knowledge.

---

## 🚀 Features

- **Natural Language Queries** – Users can ask questions in plain English.
- **Automatic SQL Generation** – Converts user queries into SQL using LLMs.
- **Database Integration** – Executes queries on a MySQL database and fetches results.
- **User-Friendly Output** – Displays results in a simple and readable format.
- **End-to-End Data Interaction** – From user input to database query execution and result retrieval.

---

## 🛠️ Tech Stack

- **Python**
- **MySQL**
- **Large Language Models (Gemini / LLM APIs)**
- **Natural Language Processing (NLP)**
